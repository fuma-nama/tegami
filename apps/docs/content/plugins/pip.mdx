---
title: pip
description: Python workspace support and PyPI publishing.
---

The pip plugin is enabled by default. It discovers workspace members from `pyproject.toml`, bumps versions, updates dependency ranges, and publishes to PyPI with `uv publish`.

## Discovery

Tegami reads the root `pyproject.toml` and expands `[tool.uv.workspace].members`. Packages need a `[project].name` and `version`.

Workspace dependencies linked through `[tool.uv.sources]` are resolved automatically.

## Configuration

```ts
const paper = tegami({
  pip: {
    updateLockFile: false,
    bumpDep: ({ kind }) => (kind === "dependencies" ? "patch" : false),
  },
});
```

| Option           | Default | Description                                   |
| ---------------- | ------- | --------------------------------------------- |
| `updateLockFile` | `true`  | Run `uv lock` after versioning                |
| `bumpDep`        | —       | Decide how to bump dependents when a package is bumped |

Default behavior:

- `dependencies` → patch bump on the dependent
- `optional-dependencies` → no automatic bump
