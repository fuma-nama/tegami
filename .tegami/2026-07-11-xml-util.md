### Add @tegami/xml-util

A new format-preserving XML document package with a `yaml`-style API (`parseDocument`, `get`/`getIn`, `setIn`/`setText`/`setAttr`, `toString`, `toJS`). The `@tegami/maven` and `@tegami/nuget` plugins now share it instead of each carrying their own XML editor.
