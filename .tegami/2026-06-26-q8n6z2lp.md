### Improve GitHub CLI network behavior

Posting pull request preview comments now stops scanning GitHub comment pages as soon as the existing Tegami comment is found, and existing GitHub releases are checked before rendering release notes.

### Fix GitHub comment updates

Updating an existing pull request preview comment now uses the repository-scoped GitHub issue comment endpoint.

### Reduce release note command work

GitHub release note generation now reuses changelog commit lookups across releases in the same publish run.

### Fix package graph cleanup

Removing package groups now clears package membership references, so later regrouping and group-aware release policies see the current graph state.

### Stabilize registry tests

Registry-dependent tests now use the existing mocked registry helper, making the package test suite reliable in offline environments.
